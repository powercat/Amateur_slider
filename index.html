amateur_slider({slider_user_ele : ".slider2", slider_pager_bool_setup : false, slider_auto_time_setup : 5000, max_width_setup : 1000})

슬라이더 옵션

slider_user_ele : "#slider"

CSS 선택자로 슬라이더 위치를 잡으시오.

데이터 형태 : String

기본값 : 없음

slider_btn_bool_setup : true or false

next, prev 버튼 유무

데이터 형태 : boolean

기본값 : true

slider_pager_bool_setup : true or false

pager 버튼 유무

데이터 형태 : boolean

기본값 : true

slider_speed_setup : 1000

슬라이더가 넘어가는 속도

데이터 형태 : Nambar

기본값 : 500(ms)

slider_auto_time_setup : 5000

슬라이더가 넘어가는 시간

데이터 형태 : Nambar

기본값 : 10000(ms)

timing_function_setup : "ease-in-out"

슬라이더 애니메이션

데이터 형태 : String

기본값 : "ease"

slider_auto_bool_setup : true or false

슬라이더 오토 유무

데이터 형태 : boolean

기본값 : true

slider_mouse_bool_setup : true or false

마우스 이벤트 on/off

데이터 형태 : boolean

기본값 : true

slider_touch_bool_setup : true or false

터치 이벤트 on/off

데이터 형태 : boolean

기본값 : true

max_width_setup : 980

최대 너비 값

데이터 형태 : Nambar

기본값 : 없음(px)

width_setup : 1980

최대 너비 값이 넘어갈 경우 너비 값

데이터 형태 : Nambar

기본값 : 없음(px)
